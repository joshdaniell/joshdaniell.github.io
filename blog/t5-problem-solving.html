JavaScript stuff for theoretical friend who's interested

There's a function called .reduce(). Bloody handy if you want to apply a function against an accumulator and each value of the array (from left-to-right) as to reduce it to a single value. Can't tell you much more than that sorry.

There's another function you'll be pleased to know about called .map(). They say this is useful to create a new array with the results of calling a provided function on every element in this array. Essentially a shortcut for creating a new list, and the list will be comprised of the output from running a program over every item in the original list.

Any then the last one I wanted to tell you about is .filter(). This functon creates a new array with all of the elements of this array for which the provided filtering function returns true. You may want to use this when you want to duplicate an array, but with only specific parts of the contents.
