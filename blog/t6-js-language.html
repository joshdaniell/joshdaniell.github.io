In the JavaScript language, describe all the use cases for each of:
  • Parentheses ():
    • Executing a function
    • Containing a list of parameters to pass to functions
    • Control the order of execution of an expression
  • Brackets []:
    • Defining an array
    • Defining an index position within an array
  • Braces {}:
    • Defining the start and end of a function
    • Separating code into blocks within a function
  • Single and double quotes '', "":
    • Containing strings
