What a responsive site is, and why responsiveness is important

Each webpage is attached to an underlying grid. This grid allows us to position content.

What mobile first design is, and why it's important

People browse the web on different devices, which have a wide range of screen sizes and functionality. Responsive webpages allow us to design a single webpage that will provide a good experience for users across a range of devices.

What frameworks are, and their pros and cons

Semantic structure is when you use specific HTML tags to help the browser (and anyone reading your code) to better understand content.

Semantic tags have meaning, as opposed to generic tags like <div>. Examples of semantic tags include <article> and <form>. These semantic tags clearly define the content for the browser and developer.
