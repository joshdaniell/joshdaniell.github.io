Welcome!

Please bear with me - this is the first webpage that I've created. It looks shoddy. But as Reid Hoffman said, if you're not embarrassed when you look back on your first iteration, you released it too late.

There's been a lot to digest in this first week's work. HTML, CSS, Git, GitHub, Command Line, Text Editors, and Bootstrap. Hope I can keep up with the pace.

Introduction

I'm doing Phase 0 of EDA but unfortunately won't be able to participate in Phase 1. We're running pretty fast at Snowball so I need to make sure I'm committing enough time to my main priority.
